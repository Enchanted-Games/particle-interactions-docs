# What is Particle Interactions?

Particle Interactions is a mod for Minecraft: Java Edition that adds new particle effects to various aspects of the game, for example when placing a block, using a flint & steel, etc.
The mod is available for the Fabric and NeoForge modloaders and is being actively maintained on Minecraft 26.1 and 26.2
